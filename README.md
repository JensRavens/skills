# skills

A collection of Claude Code skills I've built for my own workflows. Skills extend Claude with specialized knowledge and workflows — the model picks them up automatically when their description matches what you're trying to do. Install any of them via [skills.sh](https://skills.sh) and they're available across all your projects.

## expert-interview

A three-step workflow (Deep Research → Expert Interrogation → Artifact Production) that inverts the typical AI dynamic: instead of prompting the AI to generate content, the AI interviews you to extract your expertise, opinions, and creativity. The result is a text artifact that sounds like you, not like AI.

```
npx skills add jensravens/skills/expert-interview
```

Read the story behind it: [jensravens.com/expert-interview](https://jensravens.com/expert-interview/)
